**Space Explorer** is a Flutter app that lets users explore planets in our solar system. The app features **interactive 3D models**, detailed **planet statistics**, and a smooth **dark-themed UI**.

---

## Features

* Browse planets with **interactive 3D models**.
* View detailed stats like **distance from the Sun, radius, mass, gravity**, etc.
* **Smooth dark theme** for easy viewing.
* Navigate between planets easily.
