# Space Explorer

Space Explorer is a Flutter app that lets users explore planets in our solar system. The app features interactive 3D models for some planets, detailed statistics, and a smooth dark-themed UI for an engaging experience.

## Features
- Browse planets with interactive 3D models.
- View detailed stats like distance from the Sun, radius, mass, gravity, etc.
- Smooth UI with dark theme.
- Navigate between planets easily.
